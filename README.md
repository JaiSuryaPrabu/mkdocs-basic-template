# Basic Template for MKDOCS

    This **repository** provides the basic template for the documentation. It contains the following features:
        * Theme
            * Dark mode
            * Light mode
        * Navigations
            * Expand
            * Top
            * Tabs
            * Sections
        * Table of content
        * Code
            * Annotate
            * Copy
        * Search
            * Highlights
            * Suggestions
        * Header
            * Autohide
        * Footer 
            * Social links 
        * Markdown for `python`
